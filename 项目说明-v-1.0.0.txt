《Meguru's Treasure Hunt》项目说明 v1.0.0

一、这是什么游戏？

《Meguru's Treasure Hunt》是一款轻量级 2D 挖宝小游戏。

玩家需要操控巡在不同场景中移动和挖掘，根据每次挖掘后的提示，一步步推理宝藏可能藏在哪里。每次进入关卡或点击重试后，宝藏位置都会重新随机生成，所以每一局都会有一点不一样。

简单来说：这是一个“边走边试探、根据提示找宝藏”的小游戏。


二、游戏怎么启动？

如果你是从 GitHub 下载整个项目，可以直接运行这个相对路径下的 exe：

可运行版本-v-1.0.0\release\Meguru_Nene_v2.exe

双击这个 exe 就可以运行游戏。请尽量保持整个项目文件夹结构不变，不要只单独复制 exe，因为游戏还需要读取项目里的 Resources 资源目录。

如果你看到窗口标题是 Meguru's Treasure Hunt，并且任务栏图标是巡的头像图标，就说明启动的是当前版本。


三、现在可以体验哪些内容？

1. 主界面

可以从主界面进入关卡选择，也可以点击“新手教程”进入第一个关卡。

需要说明一下：目前“新手教程”还没有做成真正的分步教学，它现在更像是一个教学用普通关卡。

2. 关卡选择

目前已经有 10 个关卡入口，可以进入不同场景进行挖宝。

不同关卡有不同背景和音乐，但当前版本并不是每个关卡都有完整剧情或 CG。

3. 挖宝玩法

进入关卡后，可以用方向键移动巡，用空格键挖掘。

挖掘后，巡会告诉你离宝藏大概有多远。你需要根据提示判断方向，逐渐缩小范围，最后找到宝藏。

4. 难度体验

普通模式下挖掘次数比较充足，适合第一次试玩。

困难模式下次数更少，更适合熟悉玩法以后挑战。

困难模式不是在游戏内按钮中切换的，而是通过项目根目录里的一个小彩蛋文件触发。具体可以看《游戏攻略-v-1.0.0.txt》中的难度说明。

5. 剧情和语音

部分关卡已经加入了剧情对话、选择支、语音播放和语音回放按钮。

其中第 3 关的剧情内容相对更完整，比较适合展示当前版本的剧情系统。

6. 胜利和 CG

挖到宝藏后，部分关卡可以按 Enter 进入胜利或 CG 界面。

不过当前版本中，并不是所有关卡都有专门制作的 CG 后续内容。


四、试玩时需要知道的小提示

1. “新手教程”按钮暂时不是完整教程

它可以进入教学关卡，但还没有文字引导、操作教学和分步提示。第一次玩的话，可以配合《游戏攻略-v-1.0.0.txt》一起看。

2. 有些按钮暂时还没有功能

当前版本里，一些按钮是为了之后扩展预留的，比如“额外内容”“系统设置”等。它们可能已经显示在界面上，但还没有完整页面或实际功能。

3. 有些关卡暂时只有基础玩法

所有关卡都可以作为挖宝关卡体验，但不是每个关卡都有完整剧情、语音和 CG。

4. 如果没找到宝藏，可以放心重试

点击重试后，关卡会重新开始，宝藏也会重新随机生成。

5. Windows 可能缓存旧图标

如果资源管理器里 exe 图标看起来没变，通常是 Windows 图标缓存的问题。实际运行后，窗口标题和任务栏图标会更容易确认当前版本。


五、推荐试玩方式

如果只是快速体验，建议这样玩：

1. 先进入“选择关卡”。

2. 选择一个喜欢的场景。

3. 用方向键移动，空格挖掘。

4. 根据提示一步步靠近宝藏。

5. 挖到宝藏后按 Enter 看看有没有后续 CG 或剧情。

6. 再试试第 3 关，体验剧情、语音和回放按钮。


六、当前版本的一句话说明

v1.0.0 是一个可以用于试玩和展示的阶段版本。

它已经可以完整体验“移动、挖掘、根据提示找宝藏”的核心玩法，也加入了一部分剧情、语音、CG 和界面美化内容。

不过，它还不是最终完成版。新手教程、设置界面、额外内容、更多关卡剧情和更多 CG 仍然会在后续版本中继续完善。
